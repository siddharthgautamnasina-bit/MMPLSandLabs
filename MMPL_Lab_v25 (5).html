Firebase Setup Required

This software uses Google Firebase for real-time sync across all devices. Setup is free and takes 3 minutes. Create your project at console.firebase.google.com, then paste your config below.

1. Go to console.firebase.google.com
2. Click "Add project" → name it "MMPL-Gudur" → Create
3. Click the web icon (</>) → Register app → "MMPL Lab"
4. Copy the firebaseConfig object shown
5. Paste it below and click Connect

In Firebase console, go to Realtime Database → Rules and set read/write to true for now. You can add authentication later.
